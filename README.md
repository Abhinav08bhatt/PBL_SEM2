## 📖 About the Project

This project was developed for **PBL SEM 2** under the subject **Programming for Problem Solving**. 
It features a dual-layered architecture: a C logic engine that safely manages native file I/O operations, bridged to an interactive Python Tkinter interface. The system acts as a fully-featured ERP, handling isolated student enrollment, multi-component grading, real-time SGPA calculations, and date-stamped attendance tracking.

## ⚙️ How to Use

Getting the ERP system running on your local machine is incredibly simple. The Python script automatically handles the C-compilation for you!

1. **Prerequisites**
   Ensure you have both Python 3 and a C compiler (`gcc`) installed on your machine (works on both Linux and Windows).

2. **Launch the Application**
   Navigate to the project directory and run the main Python GUI file:
   ```bash
   python3 CODE/python_gui/app.py
   ```

3. **Start Managing**
   - Select a **Course** and **Section** from the top navigation bar and click **Load Section Data**.
   - Navigate between the tabs to manage Students, mark Attendance, input multi-component Grades, and view live Reports!
