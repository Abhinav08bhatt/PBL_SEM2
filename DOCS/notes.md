# ERP (Teacher side)

## Creating a small scaled model :

- Files : 
    - `c_core/menu.c` : contains all the other files and operates like a main menu of project
    - `c_core/student.c` : here we add(only) a student, when a student is added here it must be added in the order function files too
    - `c_core/attendance.c` : contains the number of present days of each and every single student (student gets added here when they are added using student.c)
